# Hybrid CPU Governor Auto-Installer

A blazing-fast automatic CPU governor optimizer that simulates Intel P-core and E-core behavior on ANY Linux system, even old hardware. One command. Works everywhere.

## 🚀 What This Does

Your CPU just got **smarter**. Automatically splits cores into:

- **P-Cores (Performance):** Max speed when plugged in
- **E-Cores (Efficiency):** Battery saver when unplugged

Developed for potato hardware running on 2011 i3s. Now works on ANY system.

## ⚡ Quick Start
 Just run the install.desktop after unzipping the file and DONE!

 ##Uninstallation
 just run uninstall.desktop file!!
